# IA

Ce dossier recevra les modules explicitement demandes dans les documents :

- extraction de texte depuis les CV PDF ;
- chunking structure ;
- embeddings avec Hugging Face Transformers ;
- stockage FAISS ou equivalent ;
- matching par similarite cosinus ;
- NER ;
- copilote LLM pour expliquer, reformuler, resumer et assister.
