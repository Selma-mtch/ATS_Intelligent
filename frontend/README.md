# Interface Streamlit

Interface simple pour tester le backend.

Elle permet deja :

- inscription ;
- connexion ;
- demande de droits recruteur ;
- validation des demandes cote admin ;
- deconnexion.

Les onglets CV, offres, matching et copilote sont affiches mais pas encore branches au backend.

## Lancer

```powershell
cd frontend
.\.venv\Scripts\streamlit.exe run streamlit_app.py
```
